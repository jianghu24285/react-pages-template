# React多页项目模版
- 基于create-react-app脚手架,借用脚手架优秀的地方,做最小的改动.
- 自动扫描入口js、html模版

# 技术栈
react 16 + react-router 4 + mobx 4 + webpack 3.8 + less/sass

# 运行命令
```
npm run dev / npm run start   本地运行调试(访问地址,例: localhost:3000/monkey.html)
npm run build                 打包生产代码
```
